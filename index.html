Latest Projects

My most recent project was creating the website for the EPICS ISBVI research group at Purdue University of which I am a part of. We are trying to create many technologies to modernize the world of the blind and visually impaired.

Check out the website here: Purdue EPICS ISBVI Webpage

I also created CodeView at MHacks Winter 2015 which won the Most Innovative Hack for Teaching or Learning AND the Microsoft Sponsor prize at the hackathon.

This hack allows people to do practice their technical interviewing skills on a synchronized collaborative coding platform that allows people to test code in their browser as well as communicate using VoIP. The service is completely gamified and everyone is ranked on a leaderboard that is based on a system similar to ELO.
The website is currently hosted at: www.pradyumanvig.com

For more information, click on the following links: ChallengePost | Github

In addition, I created two hacks at Wildhacks 2014 one of which won the Microsoft Sponsor Prize:

1. The first hack (which won the Microsoft Sponsor prize at the competition) was a way to send contact information by shaking hands (using Pebble smartwatches and Android). This hack can also be modified to send payment information by venmo, etc using different gestures (its something that can be done with the way we have coded this hack but it wasn't implemented at the hackathon due to the time constraint).

For more information, click on the following links: ChallengePost | Github

2. The second hack was a way to calculate if a stock price was going to go up or down the following morning using social media and stock history. It displays the three stocks most likely to go up the next day (from the database of stocks given by the Yahoo Finance API). We used the APIs and technology from Yahoo Finance, IBM Bluemix, Paypal, MailJet, and dunnhumby for this hack.
