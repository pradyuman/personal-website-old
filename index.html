My recent projects

I created two hacks at Wildhacks 2014:

1. The first hack (which won the Microsoft Azure prize at the competition) was a way to send contact information by shaking hands (using Pebble smartwatches and Android). This hack can also be modified to send payment information by venmo, etc using different gestures (its something that can be done with the way we have coded this hack but it wasn't implemented at the hackathon due to the time constraint).
For more information, click on the following links: ChallengePost Github

2. The second hack was a way to calculate if a stock price was going to go up or down the following morning using social media and stock history. It displays the three stocks most likely to go up the next day (from the database of stocks given by the Yahoo Finance API). We used the APIs and technology from Yahoo Finance, IBM Bluemix, Paypal, MailJet, and dunnhumby for this hack.
For more information, click on the following links: ChallengePost Github
